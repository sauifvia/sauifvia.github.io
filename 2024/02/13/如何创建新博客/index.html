如何创建新博客

创建Hexo项目
初始化一个新的Hexo项目：hexo init your-blog-name
进入项目目录：cd your-blog-name
部署你的博客: 在你的 Hexo 博客目录中，运行以下命令来部署你的博客到 GitHub Pages：

  • 1.hexo clean
  • 2.hexo generate
  • 3.hexo deploy
